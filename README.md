Ice Cream Parlour Website

A modern, responsive, and visually appealing Ice Cream Parlour Website built using HTML, CSS, JavaScript, and fully deployed on Vercel.
This project showcases a premium UI with smooth navigation, product sections, blogs, gallery, dine-in and delivery pages, and more.
